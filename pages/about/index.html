About Scrollmapper

About Scrollmapper.Github.io

This project is for the reading, preservation, and distribution of biblical and extra-biblical texts.

It is based on two github projects: Scrollmapper Bible Databases, and the Scrollmapper Deuterocanonical Project.

https://github.com/scrollmapper/bible_databases

https://github.com/scrollmapper/bible_databases_deuterocanonical

What is Special About Scrollmapper?

This repo, like the many others on GitHub will be kept in cold storage for future generations. This is a very unique and important event. See info here: https://archiveprogram.github.com/

This event provides us with an opportunity similar to the caves of Qumran. https://en.wikipedia.org/wiki/Qumran_Caves

Using the GitHub Archive Program we can make the lost scriptures available to future generations.

It is my goal to index all of most important Christian English translation texts that the ancient churches were familiar with. There are quite a few of them.

If you would like to get involved, please comment at the Scrollmapper GitHub repos shown above.
